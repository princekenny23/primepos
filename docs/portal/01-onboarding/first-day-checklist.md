# First-Day Success Checklist

Use this checklist for new branch rollout or new tenant onboarding.

## Account and Access
- [ ] Admin user can log in
- [ ] Outlet user can log in
- [ ] Roles and permissions are assigned correctly

## Business Setup
- [ ] Business profile is complete
- [ ] Currency is correct
- [ ] Tax settings are confirmed

## Catalog Setup
- [ ] At least 10 active products created
- [ ] Categories are set
- [ ] Unit prices are confirmed
- [ ] Opening stock entered

## POS Verification
- [ ] Test cash sale completed
- [ ] Test mobile payment sale completed
- [ ] Receipt printed or generated

## Inventory Verification
- [ ] Sale reduces stock
- [ ] Stock adjustment works
- [ ] Low stock warning appears when expected

## CRM Verification
- [ ] Test customer added
- [ ] Customer can be selected at checkout

## Storefront Verification (if enabled)
- [ ] Storefront site exists
- [ ] Allowed catalog products configured
- [ ] Public storefront displays expected products
- [ ] Test order reaches dashboard orders

## Fleet Verification (if enabled)
- [ ] Vehicle created
- [ ] Driver assigned
- [ ] First route status updated
