# How to Set Up E-commerce Storefront

## Question This Answers
How do I publish products to my online storefront and receive orders?

## Steps
1. Go to Dashboard -> Storefront -> Sites.
2. Create a site or select existing site.
3. Open Storefront -> Catalog.
4. Add products to catalog (status should show In Catalog).
5. Go to Storefront -> Settings.
6. Confirm:
- Site slug
- Outlet
- WhatsApp number
- Site active is enabled
7. Open public URL and verify products are visible.

## Order Flow
1. Customer adds products on storefront.
2. Customer places WhatsApp checkout order.
3. Order appears in Dashboard -> Storefront -> Orders.

## Expected Result
- Only allowed catalog products are visible publicly.
- New orders are visible in admin orders list.

## Troubleshooting
- Public site shows no products:
  - Add include rules/products in catalog.
- Checkout fails:
  - Check WhatsApp number and backend availability.
