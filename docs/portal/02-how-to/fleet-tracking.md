# How to Use Fleet Tracking (If Enabled)

## Question This Answers
How do I set up vehicles and track delivery progress?

## Setup Vehicle
1. Go to Fleet -> Vehicles.
2. Click Add Vehicle.
3. Enter registration number, type, and capacity.
4. Save.

## Assign Driver
1. Open vehicle details.
2. Assign driver.
3. Save.

## Track Route Status
1. Go to Fleet -> Trips/Routes.
2. Create trip with destination and expected time.
3. Update status as trip progresses:
- Assigned
- In Transit
- Delivered

## Expected Result
- Team can see vehicle, driver, and trip status in one view.

## Common Issues
- Vehicle not visible in trip form:
  - Confirm vehicle status is active.
- Driver assignment missing:
  - Check driver role/permission setup.
