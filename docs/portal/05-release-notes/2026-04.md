# Release v1.9.0 - 2026-04-03

## New Features
- Added storefront product view popup with WhatsApp product inquiry action.
- Added storefront checkout popup layout for better mobile and desktop flow.
- Added dashboard catalog management view with per-site Add/Remove controls.

## Improvements
- Improved POS cart usability with better scroll behavior in long item lists.
- Improved storefront checkout reliability by validating order reference before success handling.

## Bug Fixes
- Fixed storefront create-order server error caused by incorrect receipt number method call.
- Fixed empty catalog experience in dashboard by showing full product list with catalog status.
- Fixed storefront success banner redundancy after order creation.

## Notes for Operators
- Storefront now uses strict include rules for public product visibility.
- Ensure catalog rules are configured for each storefront site.
